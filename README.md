# mac-portfolio

## Single-Page CV

This project is a single-page HTML CV that showcases my career history, education, and skills in a clean and semantic structure.

## Project Goal

The purpose of this project is to practice building a well-structured single-page CV using only HTML. It focuses on organizing content clearly with semantic tags, while also preparing the page for future styling with CSS.

## Features

- Semantic HTML structure
- Single-page CV layout
- Sections for skills, education, and experience
- SEO meta tags in the `<head>`
- Open Graph (OG) tags for social sharing
- Favicon support

## Requirements

This project includes the following:

- Proper semantic HTML tags
- A single-page layout
- Clearly separated sections for:
  - Education
  - Skills
  - Career history
- SEO meta tags
- Open Graph tags
- A linked favicon

## What I Learned

By completing this project, I practiced:

- Structuring content with semantic HTML
- Building a clean single-page layout
- Adding SEO meta tags
- Using Open Graph tags
- Preparing an HTML document for future CSS styling
